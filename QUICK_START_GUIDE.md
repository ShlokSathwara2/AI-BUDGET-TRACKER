# 🚀 QUICK START GUIDE - Payment Reminder Features

## ⚡ 30-Second Setup

### Step 1: Add Your First Reminder (20 seconds)
```
Dashboard → Payment Reminders → Click +
Fill: Title, Amount, Due Date, Account
Click: Add Reminder ✅
```

### Step 2: Enable Notifications (5 seconds)
```
When browser asks: Click "Allow" ✅
(Or enable in browser settings)
```

### Step 3: Done! (5 seconds)
```
System will now:
✓ Check balance 1 day before due date
✓ Alert if insufficient funds
✓ Auto-pay on due date
✓ Update all reports & analytics
```

## ⚠️ Important Alerts

### Insufficient Balance Alert

**When**: 1 day before OR on due date
**Shows**:
```
⚠️ HDFC Bank has only ₹3,500
   but ₹5,000 is needed for
   Credit Card Bill payment
   due TOMORROW!

Current:  ₹3,500
Required: ₹5,000
Shortfall:₹2,000

[Dismiss] [Add Cash Now]
```

**Actions**:
- **Dismiss**: Close and handle manually
- **Add Cash Now**: Instantly adds cash to cover shortfall

## ❓ Troubleshooting

### Alert Not Appearing?
```
✓ Check browser notification permissions
✓ Ensure reminder linked to bank (not cash)
✓ Verify date is 1 day before or on due date
```

### Payment Not Processing?
```
✓ Check sufficient balance exists
✓ Verify reminder not already paid this month
✓ Wait for hourly check (or refresh page)
```

### Transaction Not in Reports?
```
✓ Check Reports date range includes today
✓ Remove any active filters
✓ Verify transaction exists in Transactions tab
```
